Anaconda Repository

Our repository features over 8,000 open-source data science and
machine learning packages, Anaconda-built and compiled for all

Conda

Conda is an open-source package and environment management
system that runs on Windows, macOS, and Linux. Conda quickly
installs, runs, and updates packages and their dependencies. It also
easily creates, saves, loads, and switches between environments
on your local computer. It was created for Python programs, but it
can package and distribute software for any language.
